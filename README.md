# 🌊 Ocean Rescue - 3D Underwater Adventure Game

## About the Game
Ocean Rescue is a 3D underwater adventure game where you play as a scuba diver collecting starfish to rescue marine life while avoiding sharks and jellyfish.

## Features
- 🎮 Full keyboard and controller support
- ⚡ 4 unique power-ups (Speed Boost, Shield, Sonar, Magnet)
- 🐠 Marine life companions (dolphins guide you, turtles are rideable)
- 🦈 Dynamic shark AI (chases on Hard difficulty)
- 💯 Combo scoring system
- 🌊 Underwater visual effects (caustics, depth fog, blue tint)

## Controls
- **WASD** - Move
- **SPACE** - Ascend
- **SHIFT** - Descend
- **Mouse** - Look around
- **Controller** - Full support!
